# Loan-Default-Risk-Analysis-
Loan Default Risk Analysis Dashboard – SQL + Power BI    This project analyzes loan portfolio risk and borrower behavior using SQL and Power BI. The objective is to identify patterns in borrower demographics, credit risk, and loan exposure to support risk monitoring and decision making.
